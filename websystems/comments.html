How I created this website

REFLECTION

I'm very proud of the website I created, as I have been doing this assignment throughout the time I was given to complete it. This website is a good representation of who I am as a person and it reflects my personality through the blue designs and intricate stars. I find the design to be appealing and minamalistic, having clean elements and void of any chaotic or clunky objects within the website. This website consists of a range of navy to pastel blues, as I find an effective website design to be calming in the eyes. This website is easily accessible and to navigate, making it suitable for mobile and desktop use.

Category

Details
Technical The website is adheres to the basic HTML structure, using tags such as nav, header, section and article. This creates a functional website that can be used in multiple devices, specifically desktop and mobile phones or ipads.

Structure

The website is seperated into different sections: Home, Past, Future and Comments, creating an accessible website that's easy to navigate.

Navigation

The navigation bar at the top of the page provides buttons for each page, enabling the user to easily navigate their way through the website. It also transforms into a menu list if the user is using a mobile device, further enhancing this website's accessibility.

Aesthetics This website utilises a specific color palette:
  • Navy
  • Indigo
  • Pastel Blue
  • Black
  • White


These colours were chosen for my website in order to create a calming visual and gradient effect, enhancing the user's experience as they navigate through the website.

Typography

This website utilises the "Poppins" font family due to its simplicity and sleek look.

Images

The high-quality images used throughout the website are utilised to engage users and enhance their experience due to the added visual appeal. The edges are smoothened out using a border in order to give them a unique and soft look.
Accessibility This website is created to be effectively accessible in order to enhance a user's experience.

Navigation Accessibility

The navigation bar is simple and effective, as the buttons have hover animations making it more satisfying to navigate through the website. It also adds some highlight on the text in order to inform the user they are clicking the button.

Responsive Design

This website's design is responsive and very adaptable, as it can be used through different devices. The text and images can turn smaller depending on the screen size, and this was achievable through media queries, which can adjust layout and font sizes. As a result, this website adheres to different users and takes into consideration the devices they use.
